# Página Web - Lash Designer 

Este projeto é uma landing page desenvolvida para uma Lash Designer. O objetivo é apresentar os serviços, a identidade visual da marca e facilitar o agendamento de clientes.

## 🎨 Identidade Visual
O design foi inspirado em uma estética sofisticada e minimalista, utilizando:
- **Cores:** Bronze metálico, Preto e Branco.
- **Tipografia:** Cinzel (Serifada) para elegância e Montserrat para legibilidade.
- **Conceitos de IHC:** Foco em contraste acessível e hierarquia visual clara.

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Estrutura semântica)
- **CSS3** (Layout com Flexbox/Grid e efeitos de gradiente metálico)
- **Figma** (Prototipagem de alta fidelidade)
- **Git/GitHub** (Versionamento e deploy)
